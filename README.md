# Student Performance Analysis and Prediction

## RD INFRO TECHNOLOGY - Data Science Internship

### Project Overview
This project focuses on analyzing student academic performance and predicting final grades using Machine Learning techniques. The Student Performance Dataset was used to perform data preprocessing, exploratory data analysis, feature engineering, model building, and evaluation.



## Objective
To predict students' final grades (G3) based on demographic, social, and academic factors using a Random Forest Regressor model.



## Dataset Information
- Dataset: Student Performance Dataset
- Total Records: 395
- Total Features: 33
- Target Variable: G3 (Final Grade)



## Tools and Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab
- GitHub



## Tasks Completed

### Task 1: Business Problem Understanding
Defined the problem statement and project objective.

### Task 2: Data Acquisition
Collected and loaded the Student Performance Dataset.

### Task 3: Data Preprocessing and Cleaning
- Checked dataset shape
- Inspected column names
- Checked missing values
- Checked duplicate records
- Verified data types
- Generated statistical summary

### Task 4: Exploratory Data Analysis (EDA)
- Distribution analysis
- Correlation analysis
- Feature visualization
- Relationship analysis

### Task 5: Feature Engineering
- Encoded categorical variables
- Prepared features and target variable
- Created machine learning dataset

### Task 6: Model Building
- Train-Test Split
- Random Forest Regressor Training

### Task 7: Model Evaluation
Evaluation Metrics:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

Model Performance:
- MAE = 1.105
- MSE = 3.487
- RMSE = 1.867
- R² = 0.830

### Task 8: Model Deployment (Future Scope)
The trained model can be deployed using Streamlit or Flask to create a web application for student grade prediction.



## Key Findings
- Previous grades (G1 and G2) are the strongest predictors of final performance.
- Absences also influence student grades.
- The Random Forest model achieved strong predictive performance with an R² score of approximately 0.83.



## Conclusion
This project demonstrates the practical application of Data Science and Machine Learning techniques for predictive analytics. The model successfully predicts student performance with high accuracy and provides insights into factors affecting academic success.
